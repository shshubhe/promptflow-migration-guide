This is an incremental walkthrough for moving an existing Prompt Flow application to Microsoft Agent Framework.
| Phase | Description |
|-------|-------------|
| **Phase 1: Audit & Map** | Understand your PF flow; map each node to a MAF concept |
| **Phase 2: Rebuild in MAF** | Re-implement the workflow using WorkflowBuilder |
| **Phase 3: Validate Parity** | Run both side-by-side and evaluate outputs |
| **Phase 4: Migrate Ops** | Replace deployment, tracing, and evaluation pipelines |
| **Phase 5: Cut Over** | Switch traffic to MAF; decommission PF resources |
